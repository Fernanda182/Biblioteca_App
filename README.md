# Sistema de Gestión de Biblioteca Digital

## Autor:
Fernanda Vaca

Proyecto desarrollado en Python aplicando Programación Orientada a Objetos y arquitectura por capas.

## Estructura
- modelos/: entidades del sistema
- servicios/: lógica de negocio
- main.py: punto de entrada

## Funcionalidades
- Añadir y quitar libros
- Registrar y dar de baja usuarios
- Prestar y devolver libros
- Buscar libros por título, autor y categoría
- Listar libros prestados por usuario

## Ejecución
python main.py
